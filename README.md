# Orvana Documentation

Welcome to the official documentation for **Orvana** — web-based radio playout software.

## What is Orvana?

Orvana is a professional radio playout and automation platform that runs on a server inside your station. Studios connect through any WebRTC-capable web browser — no installed software required.

> *Always in orbit around your station.*

## Tiers

| Tier | Price | Best For |
|------|-------|---------|
| **Core** | Free | Community, campus, and LPFM stations |
| **Pro** | $20/month | Commercial and professional broadcasters |
