# Welcome to Orvana

Orvana is a **web-based radio playout and automation platform** for terrestrial and streaming radio stations.

Run it on your own server. Access it from any browser. Rest easy knowing it's always in orbit.

## What is Orvana?

Orvana is open-source radio automation software that runs inside your station on your own hardware. Studios connect to it from any device with a modern web browser — no installed software required.

It's designed to give station operators confidence: start a show, walk away, and trust that the software will keep things running.
