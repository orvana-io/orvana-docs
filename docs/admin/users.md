# User Management

Orvana supports multiple user accounts with role-based access control.

## Roles

| Role | Access |
|------|--------|
| **Admin** | Full access including system settings |
| **Producer** | Can manage media library and scheduler |
| **Presenter** | Can operate playout and cart wall only |

## Core Limit

Core installations support up to 3 user accounts. Pro removes this limit.

## Adding Users

Users are managed from **Admin → Users → Add User**.
