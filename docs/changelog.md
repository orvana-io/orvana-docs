# Changelog

## Upcoming — v1.0.0

Initial public release.

### Core
- Full playout engine with cart wall
- Media library with artist/title/album separation
- Basic scheduler with hourly format clock
- 1 stream encoder (Icecast/Shoutcast)
- Remote studio (1 WebRTC slot)
- Soft time markers
- Basic recording
- Admin panel with user management (up to 3 users)
- Default light and dark UI templates
- Local, Google, and Microsoft authentication
- One-line Docker install

### Pro
- Split playout with shared clock synchronization
- Hard time markers + pitch-correct TCE (±4%)
- Backtiming engine
- Voice tracking
- Advanced format clock scheduler
- Traffic and spots management
- Unlimited stream encoders
- Unlimited remote studio slots with guest links
- Compliance recording
- Audio processing (EBU R128)
- AoIP support (Livewire, AES67)
- Unlimited users + granular roles
- SSO/SAML
- REST API + webhooks
- Multi-station support
- Auto-upgrades
