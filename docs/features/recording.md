# Recording

## Basic Recording (Core)

Record the main output as a single audio file.

## Advanced Recording (Pro)

:::info Pro Feature
Advanced recording requires an Orvana Pro license.
:::

- Multi-channel recording (separate tracks per source)
- Scheduled recordings
- Compliance logging (continuous 24/7 archiving)
- Automatic file naming and archiving
