# Streaming

Orvana can stream your audio to Icecast and Shoutcast servers.

## Core

One stream encoder included.

## Pro

:::info Pro Feature
Unlimited encoders require an Orvana Pro license.
:::

Configure multiple simultaneous streams with different bitrates and formats — for example, a high-quality stream for the website and a lower-bitrate mobile stream.
