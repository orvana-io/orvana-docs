# Remote Studio

Orvana's remote studio allows presenters to broadcast from anywhere using just a web browser.

## How It Works

WebRTC handles all audio in the browser — no plugins or software required. The presenter visits a URL and connects directly.

## Core

One remote studio slot included.

## Pro

:::info Pro Feature
Unlimited slots and guest links require an Orvana Pro license.
:::

- Unlimited simultaneous remote presenters
- Guest links with one-click access
- Talkback channel between studio and remote
