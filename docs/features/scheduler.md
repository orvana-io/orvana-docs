# Scheduler

The scheduler drives automated playback using format clocks.

## Format Clocks

A format clock defines what plays each hour — music rotations, sweepers, news breaks, and ad spots.

## Core Scheduler

Basic hourly clock with manual rotation management.

## Pro Scheduler

:::info Pro Feature
The advanced scheduler requires an Orvana Pro license.
:::

- Rules engine for dayparting
- Traffic and spots integration
- Automatic fill calculation
- Multi-clock support
