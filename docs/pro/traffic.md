# Traffic & Spots Management

:::info Pro Feature
Traffic management requires an Orvana Pro license.
:::

Orvana's traffic system manages commercial spots and integrates with the scheduler.

## Spot Scheduling

- Define spot breaks in format clocks
- Assign commercials to rotation groups
- Set flight dates, play counts, and daypart restrictions
- Generate traffic reports
